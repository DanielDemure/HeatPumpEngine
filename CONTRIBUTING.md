# Contributing to Hexure HPE

Thank you for your interest in contributing. We welcome contributions that improve safety, reproducibility, and clarity.

Please follow these rules:

1. Read SAFETY.md before attempting any experimental work.
2. File issues for proposed changes; label safety-related items clearly.
3. Submit pull requests against a feature branch and include tests/data where applicable.
4. For experimental contributions, include detailed provenance of any data and describe lab conditions.

We may refuse contributions that introduce unsafe or irresponsible content.
